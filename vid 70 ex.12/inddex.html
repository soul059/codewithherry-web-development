It seems like you're trying to change the color of elements with the class `.box` to a random color from an array. However, there are a few issues with your code. Here's the corrected version: ```javascript let x = Math.floor(Math.random() * 10); console.log(x); let c = ["red", "blue", "green", "white", "purple", "black", "yellow", "pink", "brown", "orange"]; let e = document.querySelectorAll(".box"); for(let i = 0; i < e.length; i++){ e[i].style.color = c[x]; e[i].style.backgroundColor = c[x]; } ``` Here's what I changed: - Fixed the typo in "yellow". - `querySelectorAll` returns a NodeList, so you need to loop through each element to apply the style. - The property for changing text color is `color`, not `Color`. - The property for changing background color is `backgroundColor`, not `backgrouncolor`.
